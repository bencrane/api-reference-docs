# Deliver Snapshot

> Deliver the snapshot content to the specified storage



## OpenAPI

````yaml dca-api POST /datasets/v3/deliver/{snapshot_id}
openapi: 3.1.0
info:
  title: Brightdata API
  description: API for interaction with datasets marketplace
  version: 1.0.0
servers:
  - url: https://api.brightdata.com
security:
  - bearerAuth: []
paths:
  /datasets/v3/deliver/{snapshot_id}:
    post:
      description: Deliver the snapshot content to the specified storage
      parameters:
        - name: snapshot_id
          in: path
          required: true
          schema:
            type: string
            example: s_m4x7enmven8djfqak
            description: The ID that was returned when the collection was triggered
        - name: notify
          in: query

            type: string

        required: true

                properties:

                    type: string
                    description: >-
                      ID of your delivery request that can be used in Monitor
                      Delivery API
````
