# Real-Time Finance Data - Introduction

## Real-Time Finance Data API Documentation

Complete reference documentation for the OpenWeb Ninja Real-Time Finance Data API. Includes code examples, data samples, and usage guides for getting quotes and trends for stocks, indices, ETFs, forex, and crypto with related news and analytics.
